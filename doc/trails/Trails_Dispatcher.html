Class Details



[line 52]

The Dispatcher is used to map an incoming HTTP request to a Controller  producing a response which is then rendered. To initialize an instance of  class Trails_Dispatcher you have to give three configuration settings:

trails_root - the absolute file path to a directory containing the                         applications controllers, views etc.            trails_uri - the URI to which routes to mapped Controller/Actions                         are appended    default_controller - the route to a controller, that is used if no                         controller is given, that is the route is equal to '/'
After instantiation of a dispatcher you have to call method #dispatch with  the request uri to be mapped to a controller/action pair.
